# 🌿 GreenRise - Nature Travel Webpage

GreenRise is a responsive nature travel webpage built using HTML and CSS. The project is designed to showcase beautiful travel destinations, nature galleries, and provide a clean and engaging user experience. It focuses on modern webpage design principles while practicing core front-end development concepts.

---

## 📖 Project Overview

GreenRise allows visitors to explore nature-inspired destinations through a visually appealing single-page layout. The webpage features a full-screen hero section, animated falling leaves, destination cards, image gallery, contact form, and footer.

This project was created to strengthen front-end development skills using HTML and CSS without relying on external frameworks.

---

## ✨ Features

- Responsive single-page layout
- Modern hero section with background image
- Animated falling leaves effect using CSS keyframes
- Navigation bar with section links
- About section introducing the GreenRise concept
- Destination cards with location icons
- Travel gallery showcasing nature experiences
- Contact form for visitor inquiries
- Footer section with copyright information
- Hover effects and smooth visual interactions
